Настройка двухэтапной аутентификации

Для активации двухэтапной аутентификации отсканируйте штрих-код с помощью Вашего мобильного приложения.

Если штрих-код не распознается, то для настройки аккаунта в Вашем мобильном приложении введите ключ вручную

G3WQ SFTO EDYJ KUOT XVCP NUVQ SUCW S44P J2PN 2IR7 2CUK G7JZ SETM 6OSJ FHWL JJZW

Сохраните секретный ключ в надежном месте для возможности восстановления доступа к аккаунту.
